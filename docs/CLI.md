# 命令行工具使用指南

## 概述

`morse-decode` 是摩尔斯电码解码器的命令行工具，支持三种输入模式：
1. **文件模式**: 从 WAV 文件解码
2. **麦克风模式**: 从麦克风录制并解码
3. **流模式**: 持续监听音频流（实时解码）

## 安装

```bash
pip install morse-decoder
```

或从源码安装：

```bash
git clone https://github.com/yourusername/morse-decoder.git
cd morse-decoder
pip install -e .
```

## 基本用法

### 查看帮助

```bash
morse-decode --help
```

## 模式 1: 文件模式

从 WAV 文件解码摩尔斯电码。

### 基础示例

```bash
morse-decode -i audio.wav -f 550
```

### 自动检测频率

如果不确定音频的频率，可以启用自动检测：

```bash
morse-decode -i audio.wav --auto-detect
```

### 使用 V3 解码器

对于高噪声环境，可以使用 V3 解码器：

```bash
morse-decode -i audio.wav -f 550 --decoder v3
```

### 启用频率跟踪

V3 解码器支持频率跟踪功能：

```bash
morse-decode -i audio.wav -f 550 --decoder v3 --freq-tracking
```

### 自定义带宽

调整解码器的带宽：

```bash
morse-decode -i audio.wav -f 550 -b 150
```

## 模式 2: 麦克风模式

从麦克风录制音频并解码。

### 基础示例

录制 5 秒音频并解码：

```bash
morse-decode --mic -d 5 -f 550
```

### 录制时长调整

录制 10 秒：

```bash
morse-decode --mic -d 10 -f 550
```

### 使用 V3 解码器

```bash
morse-decode --mic -d 5 -f 550 --decoder v3
```

### 自定义采样率

```bash
morse-decode --mic -d 5 -f 550 --sample-rate 16000
```

### 使用提示

1. 确保麦克风正常工作
2. 录制前开始播放摩尔斯电码音频
3. 在安静环境中录制以获得最佳效果

## 模式 3: 流模式

持续监听音频流，实时解码摩尔斯电码。

### 基础示例

```bash
morse-decode --stream -f 550
```

### 使用 V3 解码器

```bash
morse-decode --stream -f 550 --decoder v3
```

### 调整块大小

块大小决定了解码的频率（越小越频繁）：

```bash
morse-decode --stream -f 550 --block-size 0.5
```

### 使用提示

1. 按下 `Ctrl+C` 停止监听
2. 解码结果会在检测到有效信号时实时显示
3. 适合实时摩尔斯电码监听应用

## 参数说明

### 输入模式（三选一）

| 参数 | 简写 | 说明 |
|------|------|------|
| `--input` | `-i` | 输入 WAV 文件路径 |
| `--mic` | - | 从麦克风录制 |
| `--stream` | - | 持续监听音频流 |

### 解码器参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--frequency` | `-f` | 目标频率 (Hz) | 550 |
| `--bandwidth` | `-b` | 带宽 (Hz) | 100 |
| `--decoder` | - | 解码器类型 (v2 或 v3) | v2 |
| `--freq-tracking` | - | 启用频率跟踪 (仅 V3) | False |
| `--auto-detect` | - | 自动检测音频频率 | False |

### 录制/流参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--duration` | `-d` | 录制时长 (秒) | 5.0 |
| `--sample-rate` | - | 采样率 (Hz) | 8000 |
| `--block-size` | - | 流模式块大小 (秒) | 1.0 |

## 高级用法

### 组合参数示例

```bash
# 自动检测频率 + V3 解码器
morse-decode -i audio.wav --auto-detect --decoder v3

# 麦克风录制 + V3 + 高采样率
morse-decode --mic -d 10 --decoder v3 --sample-rate 16000

# 流模式 + V3 + 小块大小（更频繁解码）
morse-decode --stream --decoder v3 -f 550 --block-size 0.5
```

### 批量处理

使用 shell 脚本批量处理多个文件：

```bash
#!/bin/bash
for file in *.wav; do
    echo "解码 $file..."
    morse-decode -i "$file" --auto-detect
done
```

### 保存解码结果

```bash
# 保存到文件
morse-decode -i audio.wav -f 550 > result.txt
```

## 故障排除

### 错误: 文件不存在

```
❌ 错误: 文件不存在: audio.wav
```

**解决**: 检查文件路径是否正确。

### 错误: 需要安装 sounddevice

```
❌ 错误: 需要安装 sounddevice 库
```

**解决**:
```bash
pip install sounddevice
```

### 错误: 音频设备不可用

```
❌ 录制失败: [Errno -9996] Invalid input device index
```

**解决**: 检查麦克风是否正常工作，或使用文件模式。

### 解码结果为空

**可能原因**:
1. 频率设置错误
2. 音频质量太差
3. 噪声过大

**解决**:
1. 使用 `--auto-detect` 自动检测频率
2. 使用 V3 解码器
3. 在安静环境中录制

## 最佳实践

1. **文件模式**: 使用 `--auto-detect` 自动检测频率
2. **麦克风模式**: 在安静环境中录制
3. **流模式**: 根据需要调整 `--block-size`
4. **高噪声**: 使用 V3 解码器
5. **实时应用**: 使用 V2 解码器
